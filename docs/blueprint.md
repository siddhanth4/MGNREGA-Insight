# **App Name**: MGNREGA Insight

## Core Features:

- District Selection: Allow users to select a district within the chosen state (e.g., Maharashtra) to view MGNREGA performance data.
- Data Aggregation: Aggregate MGNREGA data from the data.gov.in API, including monthly performance metrics. Implement caching to handle API downtime or rate limits.
- Historical Data Display: Present historical data trends, enabling users to compare current and past performance of their district over months or years.
- Performance Metrics: Display key MGNREGA performance indicators (e.g., person-days generated, funds utilized) in an easy-to-understand format, possibly using charts or infographics.
- Comparative Analysis: Enable users to compare their district's performance against other districts within the same state to identify best practices or areas for improvement.
- Location Detection: Suggest the user's district based on their device's location, with a manual override option.
- AI-Powered Plain Language Summary: Generate a plain language summary of the district's MGNREGA performance, highlighting key achievements, challenges, and trends for users with low data literacy, incorporating tool-use reasoning for nuanced explanations.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to reflect the rural nature of the program and promote a sense of grounding and stability.
- Background color: Soft beige (#F5F5DC) to ensure readability and a gentle, non-intrusive backdrop.
- Accent color: Warm gold (#FFD700) to highlight important data points and call to action, conveying value and success.
- Body and headline font: 'PT Sans', a humanist sans-serif font combining modernity with warmth for accessibility and readability.
- Employ simple, culturally relevant icons representing different MGNREGA activities, easily understandable by a low-literacy population.
- Use a clean, uncluttered layout with ample whitespace to avoid overwhelming users and improve comprehension.
- Subtle animations for loading data or transitioning between views to provide feedback without distracting users.
